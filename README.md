1. Clone the repository
2. Copy `.env.example` to `.env` and fill in:
   - Auth0 credentials
   - Base/OnchainKit API keys
   - Contract addresses
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run development server:
   ```bash
   npm run dev
   ```
